☁️ Sincronización en la nube (Firebase)

Conecta tu cuenta de Google para que tus datos se guarden en la nube y se sincronicen automáticamente en todos tus dispositivos.

📖 ¿Cómo configurar Firebase? (paso a paso, gratis)
  1. Entra a console.firebase.google.com y crea un proyecto nuevo (es gratis).
  2. En Build → Authentication → Sign-in method, activa el proveedor Google.
  3. En Build → Firestore Database, crea la base de datos (modo producción).
  4. En Project settings (⚙️) → Your apps, crea una app Web (icono </>) y copia los valores de firebaseConfig.
  5. Pega esos valores en los campos de arriba y pulsa Guardar configuración.
  6. Pulsa Conectar con Google e inicia sesión. ¡Listo, ya se sincroniza!
  7. En Firestore → Rules pega las reglas de seguridad que están en el README para que cada usuario solo acceda a sus datos.
  8. Si publicas con GitHub Pages, agrega tu dominio (*.github.io) en Authentication → Settings → Authorized domains.
